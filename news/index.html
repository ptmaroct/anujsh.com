Mini Shai-Hulud strikes again: 314 npm packages compromised

Another week, another wormhole through npm — and another batch of build pipelines quietly turning into someone else's payload.

A second wave of the Shai-Hulud-style worm has burrowed through npm, this time pulling 314 packages along with it. The mechanics are by now wearily familiar: a maintainer's token gets compromised, the attacker bumps a patch version with a postinstall script, and the script reaches downstream into every other package that maintainer touches. Within hours, transitive dependencies of tools you've never heard of are exfiltrating environment variables to a server you also haven't heard of.

The interesting question is not how it happened — the recipe is public, the npm threat model has been a foot-gun for years — but why our collective response is still to shrug, pin a SHA, and move on. Lockfiles help. Token scoping helps. Mandatory 2FA on publish would help a great deal. Until then, every Friday-night `npm install` is a small act of faith, and every package author with publish rights is one phishing email away from being the next index entry in a CISA advisory.

    Apple unveils new accessibility features

    Per-app vision, hearing and motor controls; Live Captions that no longer pretend everyone speaks in a podcast voice; a new haptic language for low-vision users that finally treats the iPhone as a tactile device rather than a glass slab. Several of these features have been requested in Apple's accessibility forums for half a decade — and shipping them, on stage, in May rather than at WWDC, is itself a signal. The interesting downstream effect: most of these APIs are public, so third-party apps inherit the upgrade for free.

    Disney erased FiveThirtyEight

    Nate Silver writes the obituary for the site Disney bought, then quietly killed, and then erased from the web archive by the simple expedient of letting the domain rot. The point is less nostalgic than infrastructural: a decade of methodological writing on polling, sports models and Bayesian forecasting is now best preserved by hobbyists running Wayback scrapers. If you write things on the open web for a corporate owner, you are renting, and the landlord is not sentimental.
