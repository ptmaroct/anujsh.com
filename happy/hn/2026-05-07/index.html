Google Chrome silently installs a 4 GB AI model on your device without consent

“If Gemini Nano is becoming a default Chrome dependency, 'it's just a browser' stops being true.”

A privacy researcher noticed Chrome quietly downloading a multi-gigabyte Gemini Nano model — no prompt, no toggle, no clear opt-out — and the comments confirm this is shipping broadly. Storage isn't the only cost: this is Google folding a private LLM into the install footprint of the world's default browser, on machines whose owners never agreed to host one.

The defensible read is 'it enables on-device features' — and yes, local inference beats round-trips to a server farm. The harder question the thread keeps circling is consent precedent: once a 4 GB model lands without asking, the next 8 GB one doesn't need to either. For anyone shipping web apps, this also quietly turns 'does the user have an LLM' from a feature flag into an assumption.

SQLite Is a Library of Congress Recommended Storage Format

The Library of Congress added SQLite to its formal list of recommended formats for long-term preservation — the same status as PDF/A and TIFF. A quiet but important institutional vote for 'a single file, a public schema, no server' as an archival primitive. If you're hoarding research data in JSON or CSV, this is your nudge.
