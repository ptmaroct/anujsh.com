· SSH access

Give me a key, in one command.

Drop my public key on any server so I can help without passwords flying around. Safe, idempotent, and reversible.

01 · Quick install

Paste into a shell on the target machine. No side effects beyond ~/.ssh/authorized_keys.

$ curl -fsSL https://anujsh.com/ssh.sh | sh

Re-running is safe — the script checks before appending.

02 · Manual setup

If piping curl into sh is not your thing, run these three lines instead.

  1. Open a shell on the target server.
  2. Paste the block below, hit enter.
  3. Done — test with ssh user@host.
shell
1mkdir -p ~/.ssh && chmod 700 ~/.ssh 2echo "ecdsa-sha2-nistp256 AAAAE2VjZHNhLXNoYTItbmlzdHAyNTYAAAAIbmlzdHAyNTYAAABBBGdaNTFylN4rZ2OR6D326SPpEOIIoFRHsK1/jvIZjd7S9lahefifkCfQ9DoDT+AyOFBq1+yOsb5UTQoACttQ01M= #ssh.id - @ptmaroct" >> ~/.ssh/authorized_keys 3chmod 600 ~/.ssh/authorized_keys
